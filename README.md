# learning-journal-rev
# Reverse Engineering Learning Journal 🔍

Repository ini berisi catatan perjalanan belajarku dalam memahami **Reverse Engineering**.

Tujuan dari repository ini adalah sebagai dokumentasi progress belajar, mencatat hal-hal baru yang dipelajari, menyimpan hasil eksperimen, dan menjadi referensi pribadi ketika ingin mengulang materi.
---

## About This Journal

Di sini saya mencatat berbagai hal yang saya pelajari, mulai dari:

- Dasar-dasar reverse engineering
- Analisis file binary
- Static analysis
- Dynamic analysis
- Debugging
- Assembly dasar
- Memory analysis
- Malware analysis (basic)
- CTF challenges
- Tools yang digunakan
- Catatan kesalahan dan hal baru yang ditemukan
